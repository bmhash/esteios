# Sources — Reference Material Instructions

The `sources/` directory contains reference material, quotes, notes from reading, and research.

## Purpose

- **Reference library** — quotes from Meditations, Paideia, and other philosophical texts
- **Reading notes** — observations and insights from source texts
- **Cross-references** — connections between different philosophical concepts
- **Citation tracking** — keeping track of where quotes and ideas come from

## AI Role in Sources

- Help organize sources by theme, author, or concept
- Verify quotes and attributions against source PDFs in `data/`
- Format quotes for insertion into chapters
- Suggest relevant sources for specific chapter themes
- Cross-reference between different source texts
- Never invent or misattribute quotes

## Working with Sources

When adding a new source reference:
- Include full citation (author, work, book/section/page)
- Note the original language (English, Brazilian Portuguese, etc.)
- Add any necessary translation or adaptation notes
- Organize by theme or concept for easy retrieval
- Link to related chapters when relevant

## Source Texts Available

Primary sources in `data/`:
- **Marcus Aurelius - The Meditations of Marcus Aurelius.pdf** (English)
- **Werner Jaeger - Paideia.pdf** (Brazilian Portuguese)

Use `/extract-quote` workflow to find and verify passages from these texts.
