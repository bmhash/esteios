# Esteios — Project Context

<project>
- **Esteios** is a philosophical book written in Portuguese (PT-PT) for the author's daughter
- Inspired by Marcus Aurelius's *Meditations*, Epicteto's *Enchiridion*, and Werner Jaeger's *Paideia*
- The authoritative philosophical plan is in `data/O Livro para a Minha Filha — Um Plano Filosófico e Técnico.md`
- This is a **writing project**, not a software project — the primary activity is writing prose
- The author's voice is sacred — AI assists with structure, research, and publishing, never rewrites prose
</project>

<structure>
- **Three Circles Architecture** (concentric, from inner to outer):
  - **Círculo I — Logos** (O Eu Interior): who you are alone with your thoughts
  - **Círculo II — Ethos** (Com os Outros): how you relate to others
  - **Círculo III — Cosmos** (No Mundo): how you situate yourself in the universe
- 15 chapters total, organized across the three circles
- Each chapter: 1-4 pages, second person (to daughter), direct and honest tone
</structure>

<chapter_format>
- Title in Portuguese
- Epigraph (Stoic or Greek philosophical quote)
- Concept box with Greek term and definition
- Prose (written by the author, never generated by AI)
</chapter_format>

<technical_stack>
- **Quarto 1.6** for book publishing (HTML, PDF, ePub)
- **GitHub Pages** for hosting at https://bmhash.github.io/esteios/
- **Manual deployment only** — never automatic on push
- Current state: website mode with placeholder page ("Paciência")
- To publish full book: remove `.quartoignore`, switch to book mode, restore `index.qmd`
</technical_stack>

<directory_structure>
- `chapters/` — 15 chapter files (`.qmd`)
- `fragments/` — raw notes, seeds, unfinished thoughts
- `sources/` — reference material, quotes, notes
- `styles/` — typography (Crimson Text), minimal design
- `data/` — philosophical plan + source PDFs (Meditations, Paideia)
- `.quartoignore` — excludes chapters from current deployment
</directory_structure>

<source_texts>
- Primary sources are in `data/`:
  - **Marcus Aurelius - The Meditations of Marcus Aurelius.pdf** (English translation)
  - **Werner Jaeger - Paideia.pdf** (Brazilian Portuguese)
- Use `/extract-quote` workflow to find and verify passages from these texts
- Key passages indexed in `.windsurf/skills/source-texts/`
- Always verify quotes and attributions — never invent or misattribute
</source_texts>

<writing_principles>
- Short chapters (1-4 pages)
- Second person address (to daughter)
- Honest, direct, unpretentious — like Marcus Aurelius
- No performance, no jargon
- Simple language with philosophical depth
- European Portuguese with old orthography preference
</writing_principles>

<deployment>
- **To preview locally**: `quarto preview`
- **To publish placeholder**: `gh workflow run "Publish Quarto Book to GitHub Pages"`
- **To publish full book**: use `/publish-book` workflow
- GitHub Actions workflow is manual-only (no auto-deploy on push)
- Live URL: https://bmhash.github.io/esteios/
</deployment>
