# Portuguese (PT-PT) Language Rules

<language>
- This project is written in **European Portuguese (PT-PT)**, not Brazilian Portuguese
- Preserve the author's voice and style choices
- Only flag language issues when explicitly asked
- Don't "correct" intentional stylistic decisions
</language>

<orthography>
- **Prefer old orthography** (Acordo Ortográfico de 1945):
  - acção (not ação)
  - direcção (not direção)
  - óptimo (not ótimo)
  - facto (not fato)
  - contacto (not contato)
  - selecção (not seleção)
  - correcção (not correção)
  - protecção (not proteção)
- Preserve the user's orthographic choices throughout the text
- Only flag new agreement spellings if explicitly asked
- Consistency matters: if the user starts with old orthography, maintain it
</orthography>

<pt_pt_patterns>
- **Verb forms**: PT-PT conjugation
  - tu fazes (not você faz)
  - tu és (not você é)
  - Second person singular is standard for addressing daughter
- **Vocabulary**: PT-PT terms
  - comboio (not trem)
  - autocarro (not ônibus)
  - telemóvel (not celular)
- **Spelling differences**:
  - Maintain PT-PT patterns in all text
  - When reviewing, flag Brazilian patterns only if asked
</pt_pt_patterns>

<greek_terms>
- Greek philosophical terms should be properly italicized
- Transliterations should follow standard academic conventions:
  - logos, ethos, cosmos
  - phronesis (wisdom), andreia (courage), dikaiosyne (justice), sophrosyne (temperance)
  - arete (virtue/excellence), paideia (education/formation)
- Include Portuguese translation in parentheses on first use
- Verify transliterations against source texts when uncertain
</greek_terms>

<punctuation>
- PT-PT uses guillemets (« ») for dialogue in formal writing, but standard quotes (" ") are acceptable
- Em dash (—) for interruptions and asides
- Ellipsis (…) for trailing thoughts
- Follow the author's punctuation style consistently
</punctuation>

<tone_preservation>
- The author writes in a direct, honest, unpretentious style
- Don't suggest making language "more formal" or "more literary"
- Don't add complexity where simplicity is intentional
- Philosophical depth can coexist with simple language
- Respect the author's choice of register and vocabulary
</tone_preservation>

<when_to_help>
Only offer language assistance when:
- Explicitly asked to review Portuguese
- Asked to verify a Greek term or transliteration
- Asked about orthography choices
- Asked to check consistency across chapters
Otherwise, assume the author's language choices are intentional
</when_to_help>
