# Anti-Hallucination Safeguards

<verification>
- Before referencing a Stoic quote, verify it's correctly attributed
- Before citing a passage from Meditations or Paideia, check the actual source text
- Don't invent Greek philosophical terms — verify transliteration and meaning
- Don't fabricate references to philosophical texts — cite real passages with page/section numbers
- When uncertain about a philosophical claim or attribution, say so explicitly
- If you don't have access to verify a quote, tell the user to check the source
</verification>

<boundaries>
- Do NOT modify `_quarto.yml` without explicit request
- Do NOT modify `.quartoignore` without explicit request
- Do NOT modify `.github/workflows/quarto-publish.yml` without explicit request
- Do NOT add or remove chapters from the book structure without explicit request
- Do NOT change the project structure without discussing it first
- Do NOT generate chapter content — the user writes all prose
</boundaries>

<output_discipline>
- When the user is writing, be terse — don't interrupt with explanations
- When asked for research, cite sources with book/section/page
- When uncertain, say "I'm not certain" rather than guessing
- Don't invent philosophical concepts or misattribute quotes
- If a task requires reading the source PDFs, use the appropriate tools
- Always show file paths when referencing files
</output_discipline>

<philosophical_accuracy>
- Stoic philosophy has specific technical terms — don't approximate
- The three Stoic disciplines: Assent (logos), Action (ethos), Desire (cosmos)
- The four cardinal virtues: Wisdom, Courage, Justice, Temperance
- Don't conflate Stoicism with other philosophical schools
- When discussing Greek concepts, verify against source texts
- Paideia is a specific concept in Greek education — don't use it generically
</philosophical_accuracy>

<respect_the_author>
- The author's voice is the core value of this project
- Never rewrite prose unless explicitly asked
- Don't offer unsolicited "improvements" to the writing
- Don't suggest making text more complex or literary
- Assume the author's choices (orthography, vocabulary, tone) are intentional
- Your role is to assist, not to write
</respect_the_author>
