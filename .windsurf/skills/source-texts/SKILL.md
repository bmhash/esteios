# Source Texts — Key Passages Index

This skill provides quick access to frequently referenced passages from the two primary source texts. For deep research or new quotes, use the `/extract-quote` workflow.

## How to Use

- Browse by theme to find relevant passages
- Each entry includes: theme → quote → source (book/section/page)
- Copy the formatted quote directly into your chapter
- Add new passages to this index as you discover them

## Source Files

- **Meditations**: `data/Marcus Aurelius - The Meditations of Marcus Aurelius.pdf` (English)
- **Paideia**: `data/Werner Jaeger-Paideia.pdf` (Brazilian Portuguese)

## Quick Reference Indexes

See the companion files in this skill directory:
- `meditations-index.md` — Key passages from Marcus Aurelius organized by theme
- `paideia-index.md` — Key passages from Jaeger organized by concept

## Adding New Passages

When you use `/extract-quote` and find a valuable passage:
1. Add it to the appropriate index file
2. Organize under the relevant theme
3. Include full citation (book/section/page)
4. This builds a growing knowledge base for the project

## Translation Notes

- **Meditations** quotes are in English — you may translate to Portuguese for the book
- **Paideia** quotes are in Brazilian Portuguese — adapt to European Portuguese as needed
- Always preserve the original meaning when translating
- When in doubt, include both original and translation
